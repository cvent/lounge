BaseModel

BaseModel implements AbstractBaseModel and is a representation for all created Document
instances that have a user defined schema. Represents just the document data and generic properties and functions.
Clients should never have to call this directly. Inherits EventEmitter

new BaseModel()

Clients do not need to create BaseModel instances manually.
